Descrizione

string substr( string value, int start, int end )

Estrae una sotto stringa da 'value' a partire dall'indice 'start' e finendo all'indice 'end'.


Parametri

value
La stringa dalla quale estrarre la sotto stringa.

start
Indice di inizio per l'estrazione.

start
Indice finale per l'estrazione.



Valore di ritorno

Restituisce una sotto stringa di 'value' .
