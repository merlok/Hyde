Descrizione

alias dlllink( int handle, string function )

Crea una variabile di tipo alias caricando la funzione 'function' dall'handle ad una libreria di sistema precedentemente creato con la funzione dllopen.


Parametri

handle
Handle ad una libreria di sistema precedentemente creato con la funzione dllopen.

function
Nome della funzione da caricare dalla libreria di sistema.

Restituisce un alias della funzione che pu√≤ essere usato come funzione a sua volta, o 0 in caso di errore.
