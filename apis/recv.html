Descrizione

int recv( int socket, object value, [int size (opzionale)] )

Legge dal socket un oggetto in base alla tipologia del parametro ed eventualmente, nel caso in cui sia specificato, dal numero di byte richiesti.


Parametri

socket
Il socket dal quale leggere l'oggetto .

value
L'oggetto da leggere dal socket, la tipologia di lettura verrà impostata in base al tipo della variabile.

size (opzionale)
Il numero di byte da leggere dal socket, se non specificato verrà usata la dimensione di default del tipo dell'oggetto nel parametro value.



Valore di ritorno

Il numero di byte effettivamente letti o -1 in caso di errore.
