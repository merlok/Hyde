Descrizione

null println( object | string value )

Stampa su console il parametro in ingresso aggiungendo un carattere newline ( '\n' ) alla fine.


Parametri

value
Nel caso il parametro sia di una tipologia tra int, float, char o string ne stamperà il contenuto, diversamente stamperà ricorsivamente la struttura e il contenuto.



Valore di ritorno

nessuno
