Descrizione

int contains( array vettore, int indice )

Funzione che rimuove il valore all'indice specificato dal vettore.


Parametri

vettore
Array dal quale rimuovere l'oggetto specificato.

indice
Indice del valore da rimuovere dal vettore.



Valore di ritorno

Restituisce l'oggetto appena rimosso.
