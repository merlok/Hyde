Descrizione

string strreplace( string value, string replace, string with )

Rimpiazza ogni ricorrenza di 'replace' all'interno di 'value' con 'with'.


Parametri

value
La stringa nella quale eseguire l'operazione.

replace
La sotto stringa da rimpiazzare.

with
La nuova stringa da mettere al posto della sotto stringa 'replace'.



Valore di ritorno

Restituisce la stringa risultante dopo l'operazione di replace.
