Descrizione

object popen( string binary, string mode )

Lancia un processo ed inizializza un pipe handle.


Parametri

binary
Comando da eseguire all'interno del processo figlio.

mode
Modalità di accesso :

r Apre il processo in sola lettura.
r+ Apre il processo in lettura e scrittura.
w Apre il processo in sola scrittura.
w+ Apre il processo in lettura e scrittura.



Valore di ritorno

Restituisce un handle al pipe che potrà essere utilizzato con le funzioni di I/O su filesystem.
