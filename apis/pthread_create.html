Descrizione

int pthread_create( string function, [object arg, ...] )

Crea un thread e lo esegue passandogli gli argomenti specificati nella funzione.


Parametri

function
Il nome della funzione da lanciare nel thread.

arg
Gli argomenti da passare alla funzione thread.



Valore di ritorno

Restituisce l'id del thread appena creato o 0 in caso di errore.
