Descrizione

string fgets( object handle )

Legge una stringa dal file al quale punta l'handle fino a quando non trova un carattere newline o un terminatore.


Parametri

handle
L'handle precedentemente aperto tramite la funzione fopen .



Valore di ritorno

Restituisce la stringa letta dal file al quale punta l'handle fino a quando non trova un carattere newline o un terminatore.
