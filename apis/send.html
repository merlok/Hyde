Descrizione

int send( int socket, object value, [int size (opzionale)] )

Scrive sul socket un oggetto in base alla tipologia del parametro ed eventualmente, nel caso in cui sia specificato, al numero di byte richiesti.


Parametri

socket
Il socket sul quale inviare i dati .

value
L'oggetto da scrivere sul socket, la tipologia di scrittura verrà impostata in base al tipo della variabile.

size (opzionale)
Il numero di byte da scrivere sul socket, se non specificato verrà usata la dimensione di default del tipo dell'oggetto nel parametro value.



Valore di ritorno

Il numero di byte effettivamente scritti o -1 in caso di errore.
