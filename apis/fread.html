Descrizione

int fread( object handle, object value, [int size (opzionale)] )

Legge dal file puntato dall'handle un oggetto in base alla tipologia del parametro ed eventualmente, nel caso in cui sia specificato, dal numero di byte richiesti.


Parametri

handle
L'handle precedentemente aperto tramite la funzione fopen .

value
L'oggetto da leggere dal file, la tipologia di lettura verrà impostata in base al tipo della variabile.

size (opzionale)
Il numero di byte da leggere dal file, se non specificato verrà usata la dimensione di default del tipo dell'oggetto nel parametro value.



Valore di ritorno

Il numero di byte effettivamente letti o -1 in caso di errore.
