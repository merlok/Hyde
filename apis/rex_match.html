Descrizione

int rex_match( string regexp, string subject )

Esegue un match booleano in base all'espressione regolare del primo parametro usando la stringa 'subject'.
Nota : E' preferibile utilizzare l'operatore =~ per le espressioni regolari, seguendo la sintassi :

match = (subject =~ regexp)


Parametri

regexp
L'espressione regolare (sintassi PERL compliant) con la quale eseguire il match.

subject
La stringa sulla quale applicare l'espressione regolare.



Valore di ritorno

Restituisce 1 in caso di match trovati, 0 in caso contrario.
