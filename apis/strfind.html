Descrizione

int strfind( string value, string substring )

Cerca la prima ricorrenza della sotto stringa all'interno della stringa principale.


Parametri

value
La stringa all'interno della quale effettuare la ricerca.

substring
La sotto stringa da cercare.



Valore di ritorno

Restituisce l'indice della prima occorrenza della sotto stringa o -1 in caso non sia presente.
