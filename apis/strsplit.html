Descrizione

array strsplit( string value, string | char token )

Suddivide la stringa in piccole parti divise dal carattere o stringa 'token' .


Parametri

value
La stringa da dividere in parti.

token Il carattere o la stringa da considerare come un separatore nella divisione.



Valore di ritorno

Restituisce un array contenente le sotto stringhe risultati in seguito alla divisione.
