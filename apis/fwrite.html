Descrizione

int fwrite( object handle, object value, [int size (opzionale)] )

Scrive nel file puntato dall'handle un oggetto in base alla tipologia del parametro ed eventualmente, nel caso in cui sia specificato, al numero di byte richiesti.


Parametri

handle
L'handle precedentemente aperto tramite la funzione fopen .

value
L'oggetto da scrivere sul file, la tipologia di scrittura verrà impostata in base al tipo della variabile.

size (opzionale)
Il numero di byte da scrivere nel file, se non specificato verrà usata la dimensione di default del tipo dell'oggetto nel parametro value.



Valore di ritorno

Il numero di byte effettivamente scritti o -1 in caso di errore.
