int fseek( int handle, int offset, int origine )

Posiziona il puntatore nel file ad una distanza di 'offset' bytes da quella descritta da 'origine'.


Parametri

handle
L'handle precedentemente aperto tramite la funzione fopen .

offset
Numero di byte di cui spostare il puntatore.

origine

Valore di ritorno
