Descrizione

string tostring( object value )

Restituisce un oggetto di tipo string impostandone il contenuto tramite il valore passato come parametro.


Parametri

value
Un oggetto ti qualsiasi tipologia, escluse alias, array e map, del quale si vuole estrarre una rappresentazione sotto forma di stringa.



Valore di ritorno

La rappresentazione dell'oggetto sotto forma di stringa.
