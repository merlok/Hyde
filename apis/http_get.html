Descrizione

string | array http_get( string host, string uri, [map headers (opzionale)], [int full_response (opzionale)] )

Esegue una richiesta HTTP di tipo GET sull'url dell'host specificato nei parametri.


Parametri

host
Hostname al quale inviare la richiesta, se viene specificato https:// all'inizio dell'host il sistema eseguirà in automatico una richiesta HTTPS.

uri
Pagina da richiedere ed eventuali parametri della pagina.

headers (opzionale)
Tramite questo parametro si può passare alla funzione una variabile di tipo map che determina come impostare gli header http della richiesta, ad esempio:
headers = map( "User-Agent" -> "Firefox ecc ecc" );

full_response (opzionale)
Se si imposta questo parametro a 1, la funzione restituirà un array dove il primo elemento saranno gli header del response http, ed il secondo i dati veri e propri.



Valore di ritorno

Una stringa contenente il risultato della richiesta o, nel caso il parametro full_response sia impostato ad 1, un dove il primo elemento saranno gli header del response http, ed il secondo i dati veri e propri.
