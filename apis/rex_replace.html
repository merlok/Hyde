Descrizione

string rex_replace( string regexp, string subject, string replace )

Esegue una sostituzione di corrispondenze in base all'espressione regolare del primo parametro usando la stringa 'subject'.


Parametri

regexp
L'espressione regolare (sintassi PERL compliant) con la quale effettuare le sostituzioni.

subject
La stringa sulla quale applicare l'espressione regolare.

replace
Il valore da inserire nelle sostituzioni.



Valore di ritorno

Restituisce una nuova stringa risultante dalle sostituzioni effettuate con l'espressione regolare del primo parametro.
