Descrizione

float atan2( float | int y, float | int x )

Calcola l'arco tangente del valore di y/x in radianti.


Parametri

y
Il valore float o int da usare nella divisione.

x
Il valore float o int da usare nella divisione.



Valore di ritorno

Restituisce l'arco tangente del valore del valore di y/x espresso in radianti nell'intervallo [-pi,+pi].
