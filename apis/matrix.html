Descrizione

matrix matrix( int rows, int columns, ... )

Crea una variabile del tipo matrix inizializzandola con gli argomenti della funzione.


Parametri

rows
Numero di righe della matrice.

columns
Numero di colonne della matrice.

value (...)
Valori arbitrari con i quali inizializzare la matrice, il loro numero deve essere coerente con la grandezza 'rows' * 'columns'.



Valore di ritorno

Restituisce una matrice inizializzata con i valori dei parametri della funzione.
