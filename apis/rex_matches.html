Descrizione

array rex_matches( string regexp, string subject )

Esegue un estrazione di corrispondenze in base all'espressione regolare del primo parametro usando la stringa 'subject'.
Nota : E' preferibile utilizzare l'operatore =~ per le espressioni regolari, seguendo la sintassi :

corrispondenze = (subject =~ regexp)


Parametri

regexp
L'espressione regolare (sintassi PERL compliant) con la quale cercare le corrispondenze.

subject
La stringa sulla quale applicare l'espressione regolare.



Valore di ritorno

Restituisce l'array contenente le corrispondenze trovate in base all'espressione regolare nel primo parametro.
