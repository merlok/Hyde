Descrizione

array readdir( string directory, [int ricorsiva (opzionale)] )

Legge il contenuto di una directory.


Parametri

directory
Il nome della directory da leggere

ricorsiva (opzionale)
Se impostato a 1 indica una lettura ricorsiva della directory, se a 0 o omesso una lettura semplicemente iterativa.



Valore di ritorno

Un array contenente la struttura della directory letta.
