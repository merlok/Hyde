Descrizione

object fopen( string file_name, string mode )

Crea l'handle ad un file in base alla modalità di accesso impostata nel parametro 'mode'.


Parametri

file_name
Il nome del file del quale si desidera creare un handle, compreso il path.

mode
Modalità di accesso :

r Apre il file in sola lettura.
r+ Apre il file in lettura e scrittura.
w Apre il file in sola scrittura, se non esiste prova a crearlo.
w+ Apre il file in lettura e scrittura, se non esiste prova a crearlo.
a Apre il file in sola scrittura, posiziona il puntatore alla fine del file, se non esiste prova a crearlo.
a+ Apre il file in lettura e scrittura, posiziona il puntatore alla fine del file, se non esiste prova a crearlo.



Valore di ritorno

Restituisce un oggetto contenente l'handle del file o in caso di errore 0 .
